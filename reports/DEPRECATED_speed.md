## ⚠️ 본 분석의 한계 (반드시 명시)

- **단독 latency 아님**: concurrency 8 환경에서 측정. GPU 경합 포함된 sample-level wall time.
- **Judge 호출 시간 포함**: Ko-MT-Bench·LogicKor 는 매 turn OpenAI API 호출 시간 포함.

- **A100 ≠ H100 NVL**: 평가 환경(A100 80GB) ≠ 운영 환경(H100 NVL 94GB). 절대값은 운영 재현 X.

### 운영 결정 시 고려사항
- **답변 길이가 긴 모델일수록 sample 시간 길어짐** (당연) — 사용자에게 보이는 응답 시간 직접 영향
- **AWQ 4-bit 모델**(32B-AWQ)이 메모리 적고 빠를 가능성 ↑ — Tensor Core 최적화
- **FP8(35B-FP8)**: H100 native라 운영(H100 NVL)에선 더 빠를 것 — A100 측정값은 emulation 가능성
- **BF16(30B-A3B, 35B-BF16)**: 메모리 큰 만큼 inference 부담
